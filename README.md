# lhr&TSX

### 项目名称：lhr&tsx

### 项目硬件环境： linux服务器

### 项目使用工具：Idea，VsCode，Postman

### 项目描述：

创建一个私人的小型网站，里面有各种教程的分享，有主页模块，还有游戏教程，考研教程，然后计算机的学习心得分享，还有好多其他的模块

### 使用技术：

前端vue（element UI） + 后端 SpringBoot + TkMybatis（持久层框架）+ mysql（数据库） + redis（做缓存） + docker（部署服务使用）
